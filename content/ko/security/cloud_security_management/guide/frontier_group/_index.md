프런티어 그룹은 Datadog이 개발 중인 최신 AI 기능을 먼저 이용할 수 있는 자율 참여 프로그램입니다. Datadog은 자체 내부 보안팀에 안심하고 적용할 수 있는 수준으로 검증된 기능만을 활성화합니다.

## 포함된 내용 {#whats-included}

### 소유권 에이전트 {#ownership-agent}

***상태:*** *프런티어 파트너 이용 가능*<br>
특정 리소스를 소유하고 해당 리소스를 수정할 책임이 있는 사람을 자동으로 추론합니다. 데이터 소스, 지원되는 리소스 유형, 수정 워크플로우에 대한 개요는 [소유권 에이전트][1]를 참조하고 자체 데이터로 소유권 규칙을 사용자 지정하는 방법은 [소유권 기본 설정][2]을 참조하세요.

[1]: /ko/security/cloud_security_management/guide/frontier_group/ownership_agent/
[2]: /ko/security/cloud_security_management/guide/frontier_group/ownership_preferences/

### 클릭 한 번으로 CSPM 수정 {#1-click-cspm-remediation}

***상태:*** *프런티어 파트너 이용 가능*<br>
실행 중인 클라우드 리소스와 해당 리소스의 IaC 정의에 대해 사람의 개입이 필요한 경우를 제외하고 자동 수정을 수행합니다.

### 통합 탐색기 {#unified-explorer}

***상태:*** *프런티어 파트너 이용 가능*<br>
잘못된 구성, ID 위험, 공격 경로를 단일 탐색기에서 함께 보여줍니다.

### CLI 취약점 {#cli-vulnerabilities}

***상태:*** *프런티어 파트너 이용 가능*<br>
CI/CD 취약점 스캔, 에이전트 피드백을 위한 명령줄을 통합합니다.

### 컨테이너 공개 접근성 {#container-public-accessibility}

***상태:*** *출시 예정*<br>
EKS 및 런타임 데이터를 활용해 컨테이너의 공개 액세스 여부를 더욱 정확하게 평가합니다.

### 중요 자산 {#critical-assets}

***상태:*** *출시 예정*<br>
관측 가능성 데이터를 기반으로 자동으로 중요 자산을 추론하며 수동 쿼리를 통해 이를 재정의할 수 있습니다.
