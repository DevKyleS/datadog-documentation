Instrumenting {{ title $language }} functions through the Datadog CDK construct is only available for AWS CDK apps written in Typescript, Python, and Go.
The [Datadog CDK construct][1] automatically installs Datadog on your functions using Lambda layers. It configures your functions to send metrics, traces, and logs to Datadog through the Datadog Lambda Extension. ### TypeScript 1. Install the Datadog CDK constructs library For AWS CDK v1: ```sh npm install datadog-cdk-constructs --save-dev ``` For AWS CDK v2: ```sh npm install datadog-cdk-constructs-v2 --save-dev ``` 2.
